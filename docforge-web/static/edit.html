프롬프트 편집

문서 생성 시스템 지시·사용자 메시지·이미지 삽화용 영문 프롬프트 생성문을 모두 바꿀 수 있습니다. 저장 위치: docforge-web/data/user_prompts.json

문서 템플릿 — 시스템 지시 (각각 Gemini 시스템 프롬프트)

문서 생성 — 사용자 메시지 템플릿

플레이스홀더: {topic} (필수). 메인 화면의 「본문 분량」 지시가 이 메시지 뒤에 자동으로 붙습니다.

이미지 삽화 — 영문 프롬프트를 만드는 Gemini용 프롬프트

플레이스홀더: {topic}, {excerpt} (생성된 글 앞부분, 분량 설정에 따라 길이 변동), {count} (장 수)
메인에서 「이미지 생성」을 켜야 이 규칙으로 Imagen 등에 넘길 영문 문장이 만들어집니다.

저장 시 {topic} 등 포맷이 깨지면 오류가 납니다. 이미지 프롬프트는 반드시 {topic} {excerpt} {count} 를 포함하세요. 이 페이지는 http://127.0.0.1:포트/edit 로 열어야 API에 연결됩니다.
